- 概念
	- 1、要知道当前学习的这门技术是一个`什么`东西？
		- git 包含 local 和 remote 两个端。这里是指的 remote。
	- 2、学习一种语言，或者技术框架、系统 等等..要知道它`为什么`存在,以及它的作用和应用场景？
		- Why: 中心协作，或者服务器发布代码。
	- 3、最后才是想办法如何去掌握其中重要的技术以及知识点？
